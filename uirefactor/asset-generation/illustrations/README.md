# Illustration Asset Generation

Prompts para ilustraciones no-combate: estaciones del Santuario y posibles banners.

Referencia visual primaria: `uirefactor/Santuario.png`.

## Comandos

```bash
export OPENAI_API_KEY="tu_api_key"

npm run assets:illustrations:sanctuary -- --dry-run
npm run assets:illustrations:sanctuary
```

Salida:

```text
public/assets/sanctuary/stations/
```
