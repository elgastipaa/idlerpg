# Global sanctuary illustration rules

Use these rules for every Forge Light sanctuary station illustration.

Generate one isolated dark fantasy premium station/object illustration only, compatible with the visual language of `uirefactor/Santuario.png`: dark background UI, bronze-gold borders, high contrast, readable in mobile rows, ornate but not noisy. The asset should feel like a rich station icon or mini-illustration, not a full scene.

Export as a 1024x1024 PNG with real alpha transparency. The background must be actually transparent pixels around the object. Do not draw UI, labels, text, numbers, logos, watermarks, square cards, frames, inventory slots, full rooms, floors, tables, or large cast shadows. A subtle object glow is allowed if it fades into transparent pixels.

Keep the object centered, with extra transparent padding. Use readable silhouette, controlled glow, gold/bronze accents, and material detail.
